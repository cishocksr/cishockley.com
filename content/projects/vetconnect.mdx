## Project Overview

VetConnect is a full-stack web application designed to help veterans easily find and access resources, services, and community support. The platform addresses the challenge of fragmented veteran services by providing a centralized, user-friendly hub.

## Key Features

- **Resource Directory**: Searchable database of veteran services and benefits
- **User Authentication**: Secure login and profile management
- **Real-time Search**: Fast, responsive search functionality
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Technical Implementation

### Architecture

The project uses a monorepo structure with Turborepo, separating frontend and backend concerns while maintaining code sharing capabilities.

### Frontend

- **React** with **TypeScript** for type-safe component development
- **Tailwind CSS** for responsive styling
- **React Query** for efficient data fetching and caching

### Backend

- **Spring Boot** REST API
- **PostgreSQL** database with JPA/Hibernate
- **JWT** authentication
- Comprehensive test coverage (60%+)
